This app searches the OMDB API for movies. 

Each search result displays the movie poster, title year of release and a button. Clicking the button currently does nothing.

Updates to the search term should update the result list.

OMDB API: http://www.omdbapi.com/
Free API key: http://www.omdbapi.com/apikey.aspx


Instructions:

1. Open your terminal and type: `git clone https://github.com/ziran-xu/movie-finder`
This clones the repo

2. cd into the new folder and type: `npm install`
This installs the required dependencies

3. To run the React project type: `npm start`
This runs the app in development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
The page will reload when you make changes.
You may also see any lint errors in the console.

4. Inside src folder, open helper.js and replace the existing API key with your own (optional).
You can get Free API key from: http://www.omdbapi.com/apikey.aspx
